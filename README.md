### 느낀점

생각했던 것 보다 더 오래 걸려서 역시 뭐든 만들어보려면 쉽지않구나 하고 생각했다.  
그래도 과제 해보기 전보다 굉장히 많은 것을 배워서 유익한 시간이었고,  
앞으로의 공부 방향을 잡을 수 있어서 좋았다.  
기능구현 관련 여러 질문거리들이 많이 생겨서, 과제 해설을 꼭 들어보고 싶다
